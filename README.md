# 🐉 中国語練習アプリ

例文を貼り付けるだけで、3種類の練習ができる中国語学習ツールです。  
HTMLファイル1つで動作します。インストール不要。

---

## ✨ 機能

| モード | 内容 |
|---|---|
| ① 見て入力 | 中国語・ピン音を見ながらキーボード入力。まず「形」を指に覚えさせます |
| ② 穴埋め | 文の後半が空欄になります。文型のパターンを意識した練習 |
| ③ 全文入力 | 日本語訳だけを見て中国語を全文入力。定着度の最終確認 |

- 正解・不正解のフィードバックとスコア表示
- 「答えを見る」ボタンで正解を確認可能
- Enterキーで「確認 → 次へ」の操作が完結
- 例文を差し替えてすぐ別の練習セットを開始できる

---

## 🚀 使い方

### 1. アプリを開く

[👉 アプリを開く](https://mia-tinker.github.io/chinese-practice/)

### 2. 例文を入力する

アプリを開くと例文入力画面が表示されます。  
以下の形式で例文を貼り付けてください。

```
中国語の文
ピン音
日本語訳
```

例文と例文の間は **空行** で区切ります。

**入力例：**

```
教室里没有教师。
Jiàoshì li méiyǒu jiàoshī.
教室には先生はいません。

这是什么？
Zhè shì shénme?
これは何ですか。
```

### 3. 練習スタート

「▶ 練習をスタート」ボタンを押すと練習画面に切り替わります。  
①②③の順に進めると効果的です。
